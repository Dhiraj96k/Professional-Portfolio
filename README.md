A premium, high-impact digital portfolio designed to transform complex engineering into elegant digital narratives. This project focuses on high-end interactivity, sophisticated typography, and modern UI patterns.

## ✨ Core Features

### 🍱 Bento Grid Projects
A dynamic, non-linear project showcase using a sophisticated Bento Grid layout. Each card features context-aware interactions and glassmorphism styling.

### 🧲 Magnetic Interactions
Tactile UI elements (Navbar links, CTA buttons, Socials) that respond to cursor proximity with a smooth magnetic "pull" effect, powered by Framer Motion.

### 🖱️ Adaptive Custom Cursor
A high-performance custom cursor with:
- **Spring-based trail**: Fluid, organic movement.
- **Context Awareness**: Transforms into a "VIEW" reveal when hovering over project cards.
- **Glassmorphism Overlay**: Elegant hover states for interactive elements.

### 🎭 Text Masking Animations
Cinematic title reveals in the Hero section using advanced text-masking techniques for a premium "editorial" feel.

## 🛠️ Tech Stack

- **Core**: React 18 + Vite (TypeScript)
- **Styling**: Tailwind CSS (with custom design tokens)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: Shadcn UI (Foundation)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm / yarn / pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
