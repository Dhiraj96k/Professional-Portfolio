## 🌟 Overview

This project is a modern, responsive, and highly interactive single-page portfolio built for developers who want to stand out. It focuses on high-end interactivity, sophisticated typography, and modern UI patterns like dark mode glassmorphism and subtle micro-animations.

## ✨ Key Features

- **🎭 Cinematic Animations**: Staggered text reveals, animated noise grain backgrounds, and smooth scroll-spy navigation.
- **🧲 Magnetic Interactions**: Tactile UI elements (Navbar links, CTA buttons, Socials) that respond to cursor proximity with a smooth magnetic "pull" effect.
- **🖱️ Adaptive Custom Cursor**: A fluid, spring-based cursor that transforms into a "VIEW" reveal when hovering over interactive elements.
- **🧊 3D Tilt Project Cards**: A sophisticated Bento Grid layout with `preserve-3d` tilt effects that react to mouse movement.
- **📊 Animated Stats & Infinite Marquee**: Dynamic counting numbers in the About section and a smooth, CSS-based infinite scrolling marquee for supplementary skills.
- **📬 In-App Contact Form**: Built-in form with simulated API calls and elegant toast notifications via Sonner.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need Node.js installed on your machine.
- Node.js (v18 or higher)
- npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Dhiraj96k/Professional-Portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd paintpalette-folio-main
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```

## 📂 Project Structure

```text
src/
├── assets/            # Static assets (Images, Backgrounds)
├── components/        # Reusable UI Components
│   ├── ui/            # Base Shadcn UI components
│   ├── Magnetic.tsx   # Magnetic interaction wrapper
│   ├── CustomCursor.tsx
│   └── ...            # Section components (Hero, About, Projects, etc.)
├── hooks/             # Custom React hooks (e.g., use-mobile)
├── lib/               # Utility functions
├── pages/             # Route pages (Index, NotFound)
├── App.tsx            # Main application router
├── index.css          # Global styles, Design Tokens & CSS Animations
└── main.tsx           # Application entry point
```
